# proof of concept: slim + jwt + angular + satellizer

##### Node Modules
* jQuery 2.1.4
* AngularJS 1.4.8
* Angular Material 1.0.1
* Satellizer 0.13.2

##### Build Stack
* Gulp
* SASS

## Requirements
* nodejs

## Setup

Install gulp and its plugins:
```
npm install
```

Run gulp and start webserver with live-reload:
```
gulp
```
